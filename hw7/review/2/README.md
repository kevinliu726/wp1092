## How To Test
1. 先在 .env 將 mongoDB 的連結寫入
```
MONGO_URL= { your mongoDB URL }
```
2. 在 own 目錄底下開啟兩個終端機視窗，分別輸入以下指令
```
yarn start // 開啟前端
yarn server // 開啟後端
```
